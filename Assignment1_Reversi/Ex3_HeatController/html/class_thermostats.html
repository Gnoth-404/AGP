Detailed Description

The Thermostats class integrates various components like temperature sensors, display, timer, Bluetooth, heat controller, buttons, and user interface to create a complete Heat Controller system.

This class is responsible for controlling the interactions between different components to control the temperature effectively. It can control temperature adjustment,update display , and establish Bluetooth connectivity for remote control.
