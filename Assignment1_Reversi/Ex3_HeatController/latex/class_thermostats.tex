\doxysubsection{Detailed Description}
The \doxylink{class_thermostats}{Thermostats} class integrates various components like temperature sensors, display, timer, \doxylink{class_bluetooth}{Bluetooth}, heat controller, buttons, and user interface to create a complete Heat Controller system. 

This class is responsible for controlling the interactions between different components to control the temperature effectively. It can control temperature adjustment,update display , and establish \doxylink{class_bluetooth}{Bluetooth} connectivity for remote control.
